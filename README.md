# Luma Food — Catering Website

Simple static website scaffold for a catering company called Luma Food.

How to run locally

1. Open `index.html` in your browser (double-click or use a local server).

Optional: run a local static server (Python 3):

```bash
python3 -m http.server 8000
# then open http://localhost:8000
```

What is included

- `index.html` — homepage with hero, menu, services, gallery, contact form
- `styles.css` — responsive styling
- `script.js` — tiny JS for form submission simulation and sample menu download
- `assets/` — place images or logos here
